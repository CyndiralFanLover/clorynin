<h2>Data Parser</h2>
<h3>CommandOptionType</h3>

Parse Option Type String To Option Type Number
```js
function CommandOptionType(OptionType)
```
`OptionType` (String) :
- `SUB_COMMAND`
- `SUB_COMMMAND_GROUP`
- `STRING`
- `INTEGER`
- `BOOLEAN`
- `USER`
- `CHANNEL`
- `ROLE`
- `MENTIONABLE`
- `NUMBER`
- `ATTACHMENT`

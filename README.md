```js
function CommandOptionType(OptionType)
```

- `SUB_COMMMAND_GROUP`

- `INTEGER`
- `BOOLEAN`
- `USER`
- `CHANNEL`
- `ROLE`
- `MENTIONABLE`

- `ATTACHMENT`
